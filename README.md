# Superhero API test task

## Описание

Функция `find_tallest_hero(gender, has_work)` обращается к публичному API супергероев, получает список героев и возвращает самого высокого героя по заданным критериям:

- `gender`: `"Male"` или `"Female"`
- `has_work`: `True` или `False`

## Принятые допущения

### Что считается "наличием работы"
Герой считается имеющим работу, если поле `work.occupation`:

- существует
- является строкой
- не пустое
- не равно `"-"`

### Как определяется рост
Рост берется из `appearance.height`, из значения в сантиметрах, например `"203 cm"`.

## Структура проекта

- `hero_service.py` — основная функция и вспомогательная логика
- `test_hero_service.py` — тесты
- `requirements.txt` — зависимости
- `README.md` — инструкция по запуску

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск тестов

```bash
pytest -v
```

## Что покрывают тесты

- все 4 комбинации входных параметров:
    - Male / True
    - Male / False
    - Female / True
    - Female / False
- проверку корректности результата
- проверку типов и значений входных параметров
- smoke-проверку доступности API
- проверку структуры ответа API

## Важно

Тесты не используют mock-объекты и работают с реальным API, как требуется в задании.
